This repository houses all code for my submission to the Big Data Cup 2026. You have found this in its original form. I may clean up this repository as time allows, but I want to leave the repository intact as it was when I submitted the paper.

This project analyzes hockey player tracking data to model stamina and maximum-effort skating using the Big Data Cup 2026 dataset.

**Disclaimer**
I extensively used GitHub Copilot with Claude Sonnet 4.6 to generate almost all of the code found in this repository. I have reviewed all of the code that is contained and believe it is free from errors. Please contact me if you notice any mistakes.

*How to Run*
1. Download all files from https://github.com/bigdatacup/Big-Data-Cup-2026/releases/tag/Data and put them in a directory called data.
2. `pip install -r requirements.txt` (I used Python 3.11.3)
3. Run `preprocess.py`
4. Then all notebooks should run successfully.
